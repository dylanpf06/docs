## Database API Usage Limits

To ensure fair usage and system stability, the Ds Cloud Database API enforces rate limits. Violations, such as abusive usage, may result in account suspension.

### Key Points

* **Abuse Policy**: Illegal or abusive use of the API can lead to account suspension.

* **Rate Limits**: If you exceed the limit, you'll receive a `429 Too Many Requests` response. Wait **30 minutes** before retrying.

* **External Applications**: Apps hosted **outside of Ds Cloud** are limited to **300 requests per minute**.

## Best Practices

1. **Monitor Usage**: Regularly check your API usage to ensure compliance with limits.

2. **Optimize Requests**: Minimize redundant calls to stay within your plan's limits.

3. **Upgrade Plans**: If your application requires higher limits, consider upgrading to a plan that meets your needs.

By following these guidelines and understanding the restrictions, you can effectively and securely use the Ds Cloud Database API. For more information or to discuss custom plans, contact our support team.
